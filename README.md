# GCP Marker Detection and Classification

## Project Overview
This project automates the detection and classification of **Ground Control Point (GCP) markers** in aerial images. GCPs are physical markers placed on the ground before a drone flight to tie aerial imagery to real-world coordinates.  

The system performs **two tasks simultaneously**:
1. **Keypoint Localization:** Predicts the exact (x, y) pixel coordinates of the GCP marker center.  
2. **Shape Classification:** Classifies the GCP marker as one of three shapes: `Cross`, `Square`, or `L-Shaped`.
